# 🍰 CakeShop 蛋糕在线商城

> **法式烘焙 · 匠心手作 · 一键送达**

CakeShop 是一个全功能的蛋糕在线商城系统，基于 Spring Boot + Thymeleaf + MyBatis 构建。系统包含**用户端**、**管理后台**和**骑手端**三大模块，支持商品浏览、在线支付、订单管理、骑手配送、AI 智能客服等完整电商功能。

---

## 📋 功能列表

### 🛒 用户端

| 功能模块 | 功能点 |
|---------|--------|
| **商品浏览** | 全商品列表、分类筛选、关键词搜索、热销排行、新品上市 |
| **商品详情** | 多图展示、商品简介、价格、库存、收藏、评价 |
| **购物车** | 加购/减购/删除/清空、库存实时校验、合并去重 |
| **订单管理** | 下单、支付（支付宝/微信模拟）、待支付/配送中/已完成/已取消多状态 |
| **评价系统** | 评分 1-5 星、文字评价、评价展示 |
| **收藏系统** | 商品收藏/取消收藏、收藏列表 |
| **个人中心** | 个人资料编辑、订单列表、收货地址管理 |
| **AI 客服** | 集成 DeepSeek API 的智能客服「小甜」，支持商品推荐和问答 |

### 🔧 管理后台

| 功能模块 | 功能点 |
|---------|--------|
| **数据概览** | ECharts 仪表盘：用户/商品/订单/分类统计、订单状态分布、分类销售分析、库存预警 |
| **用户管理** | 用户列表、审核/冻结/解冻、角色设置（普通用户/管理员/骑手）、增删改查 |
| **商品管理** | 商品 CRUD、多图片上传（封面+详情图）、分类关联、库存管理 |
| **订单管理** | 订单列表、状态筛选、关键词搜索、订单详情、**状态流转控制**（状态机校验）、配送费设置 |
| **分类管理** | 商品分类 CRUD、二级分类（父/子分类） |
| **系统设置** | 站点名称/副标题/页脚版权/联系方式/每页条数等配置 |
| **操作日志** | 管理员操作行为审计日志 |

### 🛵 骑手端

| 功能模块 | 功能点 |
|---------|--------|
| **待接单** | 查看待配送订单、一键接单 |
| **待取货** | 已接单待取货列表、开始配送 |
| **配送中** | 正在配送的订单列表、确认送达 |
| **已完成** | 历史配送记录 |
| **收入统计** | 总收入/今日收入/总单数/今日单数、近 7 日收入趋势图 |
| **个人信息** | 个人资料编辑 |
| **消息通知** | 系统消息、标记已读/全部已读 |

## 🏁 快速开始

### 环境要求

- JDK 8+
- Maven 3.5+
- MySQL 8.0+
- Node.js (可选，用于前端构建)

### 本地运行

```bash
# 1. 克隆项目
git clone https://github.com/LozeJoe/CakeShop.git
cd CakeShop/demo

# 2. 创建数据库
mysql -u root -p -e "CREATE DATABASE IF NOT EXISTS cookieshop CHARACTER SET utf8mb4"

# 3. 导入数据
mysql -u root -p cookieshop < sql/init.sql

# 4. 修改数据库配置
# 编辑 src/main/resources/application.yml，修改 datasource 密码

# 5. 启动应用
mvn spring-boot:run

# 6. 访问
# 用户端：http://localhost:8090
# 管理后台：http://localhost:8090/admin （admin / admin123）
```

### Docker 部署

```bash
cd demo
docker-compose up -d
```

访问 http://localhost:8090
