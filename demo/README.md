# 🍰 CakeShop 蛋糕在线商城

> **法式烘焙 · 匠心手作 · 一键送达**

CakeShop 是一个基于 Spring Boot + Thymeleaf + MyBatis Plus 构建的全功能蛋糕电商系统。覆盖用户选购下单、管理员数据运营、骑手接单配送三大核心场景，实现电商全链路闭环。

---

## ✨ 功能模块

### 🛒 用户端

| 模块 | 功能点 |
|------|--------|
| **商品浏览** | 全商品列表、分类筛选、关键词搜索、热销排行、新品上市 |
| **商品详情** | 多图轮播、商品简介、价格、库存、收藏、用户评价展示 |
| **购物车** | 添加/删除/修改数量/清空、库存实时校验、同商品自动合并去重 |
| **订单管理** | 创建订单、支付宝/微信模拟支付、取消订单（恢复库存）、再来一单 |
| **订单追踪** | 六状态流转：待支付→待配送→待取货→配送中→已送达/已取消 |
| **评价系统** | 1-5 星评分 + 文字评价，已送达订单可评价 |
| **收藏系统** | 商品收藏/取消收藏、收藏列表 |
| **个人中心** | 个人资料编辑、密码修改（BCrypt 加密）、省市区三级联动地址 |
| **💬 联系骑手** | 配送中订单可实时与骑手对话，消息即时双向同步 |
| **🤖 AI 客服** | 左下角「小甜」智能客服，DeepSeek API 驱动，可拖拽，支持商品问答 |
| **🗺️ 订单地图** | Leaflet + OpenStreetMap 展示收货地址定位 |
| **🌙 深色模式** | 一键切换，CSS 变量全覆盖 |

### 🔧 管理后台

| 模块 | 功能点 |
|------|--------|
| **📊 数据仪表盘** | 8 个 KPI 卡片（总营收/今日营收/订单数/待处理/用户数/商品数/骑手数/客单价） |
| | 6 个 ECharts 图表：月度销售趋势、订单状态分布饼图、热销商品 TOP10、近7天订单趋势、近30天营收趋势、分类销售玫瑰图 |
| | 30 秒自动刷新 + 手动刷新按钮 + 数字滚动动画 |
| **👥 用户管理** | 用户列表/搜索、添加/编辑/删除、审核通过、冻结/解冻、角色分配（普通用户/管理员/骑手） |
| **🎂 商品管理** | 商品 CRUD、多图片上传（封面+详情图）、库存管理、分类关联、销量统计 |
| **📦 订单管理** | 订单列表、状态筛选、关键词搜索、订单详情、配送费设置、**状态流转控制**（状态机校验） |
| **🏷️ 分类管理** | 商品分类 CRUD、父/子分类树结构 |
| **⚙️ 系统设置** | 站点名称、首页标题/副标题、每页条数、页脚版权、联系方式等全局配置 |
| **📋 操作日志** | 管理员全部操作自动记录、分页查询、每日凌晨 3:00 自动清理 30 天前日志 |
| **⚠️ 库存预警** | 自定义阈值，低库存商品弹窗提醒，断货/低库存分色标记 |

### 🛵 骑手端

| 模块 | 功能点 |
|------|--------|
| **配送管理** | 待接单列表 → 一键接单 → 待取货 → 开始配送 → 配送中 → 确认送达 |
| **状态保护** | 每步流转有状态机校验，禁止非法跳转 |
| **💰 收入统计** | 账户余额、累计总收入、累计完成单数、今日订单数、今日收入 |
| | 近 7 天每日收入明细列表 |
| **💬 与客人对话** | 配送中/待取货订单可进入聊天页，与用户实时双向对话 |
| **📬 消息通知** | 系统消息/订单通知/收入到账，已读/未读标记，全部标为已读 |
| **👤 个人信息** | 姓名/手机/邮箱/地址/密码编辑 |
| **📱 移动端适配** | 底部 Tab 导航栏，适配手机操作 |

### 后端

| 技术 | 版本 | 用途 |
|------|------|------|

| Maven | 3.5+ | 构建管理 |
| JaCoCo | 0.8.7 | 测试覆盖率 |
| JUnit 5 | — | 测试框架 |
| BCrypt | — | 密码加密（替代 MD5） |

### 前端

| 技术 | 用途 |
|------|------|
| HTML5 + CSS3 | 法式烘焙美学自研设计体系 |
| JavaScript + jQuery | 交互逻辑、AJAX 局部刷新 |
| Thymeleaf | 服务端模板渲染 |
| GSAP 3 + ScrollTrigger | 页面滚动入场动画、Hero 动画 |
| ECharts 5 | 管理后台数据可视化仪表盘 |
| Leaflet + OpenStreetMap | 订单地图定位 |
| 响应式设计 | 桌面端 / 平板 / 手机三端适配 |
| CSS 变量 + 深色模式 | 一键切换日/夜间主题 |

### 安全

| 措施 | 说明 |
|------|------|
| BCrypt 密码加密 | 自适应哈希 + 随机盐，替代不安全的 MD5 |
| 旧密码自动升级 | MD5 密码首次登录自动迁移为 BCrypt |
| XSS 防护 | 输入净化，过滤 HTML/脚本标签 |
| SQL 注入防护 | MyBatis `#{}` 参数绑定 |
| 登录拦截器 | 未登录访问受限页面跳转登录 |
| 管理员权限守卫 | 非管理员访问 `/admin/**` 返回 403 |
| 订单状态机 | 用户/管理员双流转表，终态不可回退，12 条非法路径测试覆盖 |
| CSRF 防护 | Session 级 CSRF Token |
| CSP 安全头 | Content-Security-Policy 限制脚本来源 |
| 密码强度校验 | 注册/修改密码要求大写+小写+数字+特殊字符 ≥ 8 位 |

## 🚀 快速开始

### 环境要求

- JDK 8+
- Maven 3.5+

### 方式一：H2 内存数据库（零依赖，推荐开发）

```bash
cd demo
mvn spring-boot:run -Dspring-boot.run.profiles=dev
```

启动后访问 http://localhost:8090，H2 控制台 http://localhost:8090/h2-console

### 方式二：Docker Compose（MySQL，推荐生产）

```bash
cd demo
docker-compose up -d
```

访问 http://localhost:8090

## 📂 项目结构

```
demo/
├── src/main/java/com/
│   ├── controller/
│   │   ├── AdminController.java        # 管理后台
│   │   ├── AiChatController.java       # AI 客服 API
│   │   ├── CartController.java         # 购物车
│   │   ├── GoodsController.java        # 商品展示
│   │   ├── OrderController.java        # 订单 + 用户聊天
│   │   ├── ReviewController.java       # 商品评价
│   │   ├── RiderController.java        # 骑手端 + 聊天
│   │   ├── StatisticsController.java   # 数据统计 API
│   │   └── UserController.java         # 登录注册 + 三端分流
│   ├── service/                        # 业务逻辑层
│   │   ├── OrderServiceImpl.java       # 订单状态机（6状态 · 双流转表）
│   │   ├── RiderChatService.java       # 骑手-用户聊天
│   │   └── ...                         # 10+ Service
│   ├── mapper/                         # MyBatis 持久层（12 Mapper）
│   ├── javaBean/                       # 实体类（12 个）
│   └── config/
│       ├── GlobalExceptionHandler.java  # 全局异常处理
│       ├── MybatisPlusConfig.java       # 分页插件
│       ├── LoginInterceptor.java        # 登录拦截
│       ├── AdminInterceptor.java        # 管理员权限守卫
│       ├── AdminLogAspect.java          # AOP 操作日志
│       ├── GlobalEncodingFilter.java    # 编码+安全头+CSRF
│       ├── DataInitConfig.java          # 数据初始化+密码升级
│       ├── MyMVCConfig.java             # MVC 视图+拦截器+国际化
│       └── SystemConfigService.java     # 系统配置
├── src/main/resources/
│   ├── templates/                      # 40+ Thymeleaf 模板
│   │   ├── admin/                      # 管理后台模板
│   │   ├── rider/                      # 骑手端模板
│   │   └── user/                       # 用户端公共组件
│   ├── static/                         # CSS / JS / 图片 / 字体
│   ├── i18n/                           # 中英文国际化
│   ├── application.yml                 # 主配置
│   ├── application-dev.yml             # H2 开发环境配置
│   ├── schema-h2.sql                   # H2 建表脚本
│   └── data-h2.sql                     # H2 种子数据（15商品+20订单）
├── docs/
│   └── cakeshop-roadshow.html          # 路演 PPT（GSAP 动画 + 法式配色）
├── sql/                                # MySQL 初始化脚本
├── docker-compose.yml                  # Docker 部署
└── pom.xml
```
