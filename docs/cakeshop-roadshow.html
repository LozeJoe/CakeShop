🎯 项目简介

一个完整的蛋糕在线商城系统，覆盖电商全链路

📖 项目概述

CakeShop 是一个基于 Spring Boot 技术栈构建的全功能蛋糕在线商城系统，涵盖用户购物、后台管理和骑手配送三大核心模块。

🛒

用户端

商品浏览、搜索、购物车、下单支付、评价收藏、AI 智能客服

🔧

管理后台

ECharts 数据仪表盘、用户/商品/订单管理、系统配置、操作审计

🛵

骑手端

接单配送、状态流转、收入统计、消息通知、7 日收益趋势

🧩 问题与解决

开发过程中的挑战与解决方案

🔑 关键问题与解决方案

🔐 MD5 → BCrypt

无盐 MD5 替换为自适应哈希+随机盐，旧用户自动迁移，密码强度校验（大写+小写+数字+符号≥8位）

📋 订单状态机

用户/管理员双流转表，终态不可回退，34 个测试覆盖全部路径，禁止非法状态跳转

💰 配送费修复

优先读取管理员设置的 commission，未设置时回退按订单金额 10% 自动计算

🔍 搜索翻页

搜索结果翻页链接添加 keyword 参数，保持搜索上下文不丢失

🛒 购物车去重

取消订单时检查购物车已有商品，存在则累加数量而非新建重复记录

📝 注册校验

手机号11位正则校验、邮箱格式验证、密码强度实时指示器、省市区三级联动
