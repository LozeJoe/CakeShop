📖 项目概述

CakeShop 是一个基于 Spring Boot 技术栈构建的全功能蛋糕在线商城系统，涵盖用户购物、后台管理和骑手配送三大核心模块。

🛒

用户端

商品浏览、搜索、购物车、下单支付、评价收藏、AI 智能客服

🔧

管理后台

ECharts 数据仪表盘、用户/商品/订单管理、系统配置、操作审计

🛵

骑手端

接单配送、状态流转、收入统计、消息通知、7 日收益趋势

🧩 问题与解决

开发过程中的挑战与解决方案

🔑 关键问题与解决方案

❌ 问题：订单状态无校验

管理员可以随意跳转状态，已送达可改回待支付，已取消可重新激活

✅ 解决方案

设计完整状态机，用户/管理员双流转表，34 个测试覆盖所有路径

❌ 问题：骑手配送费硬编码

骑手收入恒为订单 10%，管理员设置的配送费不生效

✅ 解决方案

改为优先读取 order.commission，兼容未设置时回退 10%

❌ 问题：搜索翻页失效

搜索结果翻页后跳回全量商品列表

✅ 解决方案

分页链接添加 keyword 参数，保持搜索上下文

❌ 问题：取消订单购物车重复

取消订单后商品重新加入购物车，多次取消产生重复记录

✅ 解决方案

合并逻辑：检查购物车是否已存在，存在则累加数量而非新建
