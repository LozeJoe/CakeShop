## 替换方式

打开对应页面 → 按 **Win+Shift+S** (截图工具) 或 **Print Screen** → 保存为 PNG → 覆盖对应文件

## 操作步骤

1. 启动应用: `cd demo && mvn spring-boot:run`

3. 依次打开上述页面
4. 截图后保存为同名 PNG 文件覆盖到 `screenshots/` 文件夹
5. 然后告诉我「截图已替换」，我来更新 GitHub 仓库
